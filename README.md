# Exercice : Bataille Navale 

Partir de l'exercice réalisé précédemment en utilisant les possibilités de la programmation orientée objet.

La grille de jeu virtuelle est composée de 10 x 10 cases.

Une case est identifiée par le joueur par une lettre identifiant la colonne (de 'A' à 'J') et un numéro de ligne (ex. : 'B2').

Le programme affiche au départ une grille vide,   
- puis demande à l'infini au joueur les coordonnées d'un tir,   
- puis indique si ce tir touche un navire (précisant si le navire est coulé à la suite de plusieurs tirs convergents, et si la partie est finie lorsque le dernier navire est
coulé),   
- puis affiche la grille actualisée suite à ce tir, sur laquelle le joueur est en mesure d'identifier   
  - les cases ayant fait l'objet d'un tir raté   
  - et celles constituant une partie d'un navire touché ou coulé (en distinguant les deux cas).
